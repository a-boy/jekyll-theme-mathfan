## About This Theme

**mathfan** is a Jekyll theme for mathematics blogs with support for:

### Math Rendering
- **MathJax**: Traditional LaTeX math rendering
- **KaTeX**: Fast KaTeX rendering

### Interactive Tools
- **SageCell**: Run SageMath code in the browser
- **Mermaid**: Draw diagrams and flowcharts
- **GeoGebra**: Interactive geometry tool

### Slide Decks
- **Marp**: Markdown presentation ecosystem
- **Reveal.js**: HTML presentation framework

## Usage

Add features to your page front matter:

```yaml
---
title: "My Post"
features:
  - katex
  - mermaid
  - sagecell
---
```
